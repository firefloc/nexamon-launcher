# Nexamon Launcher

Minecraft launcher for the Nexamon modpack.

## Download

Go to [Releases](https://github.com/firefloc/nexamon-launcher/releases/latest) to download the latest version for your platform.

| Platform | File |
|----------|------|
| Linux | `.AppImage` / `.deb` / `.rpm` |
| Windows x64 | `.exe` (NSIS installer) |
| Windows ARM | `.exe` (NSIS installer) |
| macOS x64 | `.dmg` |
| macOS ARM (Apple Silicon) | `.dmg` |

The launcher auto-updates after installation.
